# 🏨 Hotel Booking Data Analysis

## 📌 Project Overview
This project analyzes hotel booking data from an online travel platform to identify key trends, customer behavior, cancellation patterns, and business opportunities.

The goal of this project is to generate actionable insights that can help improve booking performance, reduce cancellations, and increase profitability.

---

## 📂 Dataset Information
The dataset contains **30,000 hotel booking records** with information related to:

- Customer details
- Hotel properties (city, star rating, room type)
- Booking channels (Web, Mobile App, Travel Agent)
- Pricing (booking value, cost price, selling price, cashback, coupons)
- Booking status (Confirmed, Cancelled, Failed)

---

## ⚙️ Tools & Technologies Used

- **Python** (Pandas, Matplotlib, Seaborn) – Data Analysis & Visualization  
- **SQL (MySQL)** – Data Querying  
- **Power BI** – Dashboard Creation  
- **Jupyter Notebook** – EDA & Analysis  

---

## 📊 Key Analysis Performed

### 🔹 Exploratory Data Analysis (EDA)
- Data cleaning and preprocessing
- Missing value handling
- Feature engineering (stay length, booking month)

### 🔹 Booking Trends Analysis
- Monthly booking trends
- Top cities by bookings
- Stay length distribution

### 🔹 Customer & Booking Behavior
- Booking patterns across channels
- Room type preferences
- Star rating vs booking value

### 🔹 Cancellation Analysis
- Overall cancellation rate (~20%)
- Cancellation by booking channel

### 🔹 Profitability Analysis
- Profit per booking
- Profit by room type

### 🔹 Promotions Analysis
- Coupon usage
- Cashback distribution

---

## 📈 Key Insights

- Web channel generates the highest number of bookings.
- Standard rooms are the most preferred by customers.
- Higher star-rated hotels have higher booking values.
- Around **20% of bookings are cancelled**.
- Travel agent bookings have **high-value outliers**.
- Booking demand is mostly stable, with a spike in **April**.
